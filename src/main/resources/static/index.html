Shorten a Link

API

Request
curl -X POST https://jjalb.ddmz.org/api/shorten \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/very/long/path"}'
Response
{
  "code": "aBcDeF",
  "shortUrl": "https://jjalb.ddmz.org/aBcDeF",
  "originalUrl": "https://example.com/very/long/path",
  "author": null
}
