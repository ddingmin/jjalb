MCP Server

jjalb provides an MCP (Model Context Protocol) server that allows AI agents to shorten URLs and retrieve click statistics through tool calls.

MCP is an open protocol that standardizes how AI applications interact with external tools and data sources. Compatible clients include Claude Desktop, Cursor, Windsurf, and other MCP-enabled applications.

Endpoint

Connect your MCP client to the SSE endpoint:

https://jjalb.ddmz.org/sse

Available Tools

shortenUrl

Shorten a URL and return the short link.

url (string, required) — The URL to shorten
author (string, optional) — Author name for the link

getStats

Get click statistics for a short code. Returns total clicks, daily clicks, top referrers, and top user agents.

code (string, required) — The short code to look up

Client Configuration

Claude Desktop

Add to ~/Library/Application Support/Claude/claude_desktop_config.json (macOS) or %APPDATA%\Claude\claude_desktop_config.json (Windows):

{
  "mcpServers": {
    "jjalb": {
      "url": "https://jjalb.ddmz.org/sse"
    }
  }
}
Cursor

Add to .cursor/mcp.json in your project root:

{
  "mcpServers": {
    "jjalb": {
      "url": "https://jjalb.ddmz.org/sse"
    }
  }
}
Claude Code

Run the following command:

claude mcp add jjalb https://jjalb.ddmz.org/sse

Example

Once configured, you can ask your AI agent:

"Shorten this URL: https://example.com/very/long/path"
"Show me the click stats for code aBcDeF"
