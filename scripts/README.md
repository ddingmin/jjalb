# Raycast Script Command

Raycast Script Commands for shortening URLs with [jjalb](https://jjalb.ddmz.org).

| Script | Description |
|---|---|
| `shorten-url.sh` | Enter a URL to shorten |
| `shorten-clipboard-url.sh` | Shorten the URL already in your clipboard |

## Setup

1. Download the scripts to a local directory.

2. Make it executable:

```bash
chmod +x shorten-url.sh
```

3. Open Raycast `Settings` (`Cmd + ,`) > `Extensions` > `Script Commands` > `Add Directories` and add the directory containing the script.

## Usage

1. Open Raycast (`Opt + Space`) and search for **Shorten URL**.
2. Enter the URL you want to shorten and press `Enter`.
3. The shortened URL is copied to your clipboard.
